    approach: I helped run more interactive sessions using quizzes, live examples
      and real questions people needed to answer. I also worked with developers,

    approach: I helped analyse the stock by saleability, certification risk and

    approach: I helped coordinate actions across different teams, asked questions to

    approach: I supported monthly CSAT reporting and helped launch an internal

  - title: Building an AI pixel furniture app

      grid.
    challenge: I wanted to build something that used AI in a practical, visual way,

      hard bit has been keeping the original object recognisable while still
      turning it into a clean pixel-art asset that works inside the app.
    approach: "I’ve been building the app around a local ComfyUI setup, using
      image-to-image generation rather than starting from text alone. I’ve
      worked through the messy bits: getting ComfyUI connected, testing prompts,

    outcome: "The project is still in progress, but it has helped me learn a lot by
      actually building: local AI workflows, prompt testing, image generation
      logic, frontend UX, backend connection issues and how to turn a loose idea
